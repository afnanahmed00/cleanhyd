# Cleanhyd – Cleaning Service Website

## Project Overview

Cleanhyd is a modern, responsive single-page marketing website created for a fictional residential and commercial cleaning service company. The goal of the project was to design a professional, trustworthy, and visually engaging experience that helps potential customers learn about services, compare pricing plans, and book a cleaning appointment.

The website was built from scratch using HTML, CSS, and JavaScript, with a strong focus on clean UI design, responsive layouts, and user experience.

---

## Brand Rationale

I chose the name **Cleanhyd** to create a simple and memorable brand identity. The design uses a combination of teal, white, and yellow tones to communicate cleanliness, trust, freshness, and professionalism.

The visual style is modern and approachable, making the service feel reliable while maintaining a friendly appearance for residential customers.

---

## Color System

### Primary Color

* Teal (#14B8A6)
* Used for buttons, icons, highlights, and call-to-action sections.

### Accent Color

* Yellow (#F4C430)
* Used to draw attention to important actions and featured pricing plans.

### Neutral Colors

* White (#FFFFFF)
* Dark Navy (#10233F)
* Gray (#64748B)

These colors were chosen to create strong contrast while keeping the interface clean and easy to read.

---

## Typography

The website uses **Poppins** as the primary typeface.

Reasons for choosing Poppins:

* Modern and professional appearance
* Excellent readability on desktop and mobile devices
* Works well for both headings and body text

## Responsive Design

The website was designed to work across multiple screen sizes, including:

* Desktop
* Laptop
* Tablet
* Mobile Devices (360px and above)

Responsive layouts were implemented using CSS Grid, Flexbox, and media queries.

---

## Interaction I'm Most Proud Of

The interaction I am most proud of is the responsive user flow between the landing page and the booking page.

Users can explore services, compare pricing plans, and seamlessly proceed to the booking form without leaving the visual identity of the brand. The experience remains consistent across desktop and mobile devices, creating a smooth customer journey.
